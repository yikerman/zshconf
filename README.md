# Usage

`bootstrap.sh` symlinks files in `home/` to the user's home directory;
Install these before running `bootstrap.sh`.

## Required Native Dependencies Before `bootstrap.sh`

- zsh (shell)
- git
- [starship](https://github.com/starship/starship)
- [zoxide](https://github.com/ajeetdsouza/zoxide)
- [fzf](https://github.com/junegunn/fzf)
